# Model 1 v1.0 Intro Tutorial

> Update date: 2026-06-06
> Scope: first-person stop-loss guardrail only

## Purpose

Model 1 v1.0 is a minimal stop-loss guardrail. It helps you check before action: proceed, slow down, freeze, or exit.

## Three Questions

1. Will this action create real-world consequences?
2. Have the key facts been verified from the actual source?
3. If I continue, can I still safely stop or roll back later?

## Signals

- Green: proceed.
- Yellow: slow down and verify.
- Red: freeze immediately
