# Model 1 v1.0 入门教程

> Update date: 2026-06-06
> Scope: first-person stop-loss guardrail only

## 用途

Model 1 v1.0 是一个最小止损护栏。它只帮你在行动前检查：继续、放慢、冻结，还是退出。

## 三问

1. 这个动作会不会产生现实后果？
2. 关键事实是否已从真实来源核对？
3. 如果继续，后面还能不能安全停止或回滚？

## 信号

- Green：可以继续。
- Yellow：先放慢，补验证。
- Red：立刻冻结，不执行。

## 边界

不分析别人，不操控别人，不替你做最终决定。
